## English (primary locale: `en`)

### Short description (max 132 characters)

```
Add text watermarks to PDFs entirely in your browser. Files never upload. Supports English and Simplified Chinese.
```

(118 chars)

### Detailed description

```
PDF Watermark adds text watermarks to your PDFs without uploading a single byte to any server.

Everything happens locally in your browser:
• Open a PDF (drag-and-drop or pick a file)
• See a live preview with the watermark already applied
• Adjust opacity, rotation angle, font size, and color
• Choose a layout: single (centered), 2×2 grid, or 3×3 grid
• Download the watermarked PDF

PRIVACY FIRST
The extension declares ZERO permissions in its manifest. There is no background page, no analytics, no telemetry, and no network requests of any kind. Your PDFs and watermark text never leave your device.

LANGUAGE SUPPORT
• English / Latin alphabet — uses the standard Helvetica font (zero embedding overhead)
• Simplified Chinese (简体中文) — uses an embedded subset of Noto Sans SC, covering all GB2312 Level 1 + Level 2 characters (~6,763 chars)
• Mixed Chinese + English — automatically detected per watermark

USE CASES
• Marking confidential or draft documents
• Branding invoices, contracts, or proposals
• Adding "Sample" or "Specimen" stamps to demos
• Tagging documents with company or department names

LIMITATIONS
• Maximum file size: 100 MB
• Password-protected PDFs must be unlocked before processing
• Traditional Chinese, Japanese kana/kanji, and Korean Hangul are not currently supported

Open source. Built with pdf-lib, pdf.js, and Noto Sans SC (SIL Open Font License).
```

## Simplified Chinese (locale: `zh_CN`)

### Short description (短描述, max 132 characters)

```
浏览器内本地为 PDF 添加文字水印。文件不上传任何服务器，支持中英文混排，开箱即用。
```

(40 chars — well under limit)

### Detailed description (详细描述)

```
PDF Watermark 是一个完全在浏览器本地运行的 PDF 水印工具，所有操作都不会向任何服务器上传字节。

工作流程：
• 拖拽或选择本地 PDF
• 实时预览首页带水印的效果
• 调整透明度、旋转角度、字号、颜色
• 选择布局：单点居中 / 2×2 网格 / 3×3 网格
• 下载带水印的 PDF

隐私优先
扩展的 manifest 中权限列表为空（permissions: []），没有 background 脚本、没有埋点、没有任何网络请求。你的 PDF 文件和水印文字永远不会离开本地设备。

语言支持
• 英文 / 拉丁字母 —— 使用 PDF 标准 Helvetica 字体，零嵌入开销
• 简体中文 —— 内置思源黑体子集，覆盖 GB2312 一级 + 二级共约 6763 个常用汉字
• 中英文混排 —— 按水印文字自动检测使用对应字体

适用场景
• 标记机密文件、草稿、内部资料
• 给发票、合同、提案打公司或部门标识
• Demo 文档加「样品」「示例」字样
• 文档归档分类

已知限制
• 单文件最大 100 MB
• 加密 PDF 需先解除密码
• 暂不支持繁体中文、日文假名汉字、韩文谚文

开源构建：pdf-lib、pdf.js、思源黑体（SIL OFL 字体许可证）。
```
